# 🎤 스피킹 만능문장 연습

스피킹 만능문장을 암기·발음 연습할 수 있는 웹앱입니다.
설치 없이 브라우저로 접속해 휴대폰·데스크탑 어디서나 사용할 수 있습니다.

> **이 앱에는 문장 데이터가 포함되어 있지 않습니다.** 저작권 보호를 위해, 본인이 가진
> 텍스트 파일을 직접 불러와 연습하는 방식입니다. 불러온 내용은 서버로 전송되지 않고
> **사용하는 기기의 브라우저에만** 저장됩니다.

## 사용법
1. 사이트 접속 → **📁 텍스트 파일 선택** 또는 텍스트 **붙여넣기**로 문장을 불러옵니다.
2. 한 번 불러오면 그 기기에서는 다음 접속 때도 유지됩니다. (상단 📁 버튼으로 교체/삭제)

## 지원 파일 형식
아래 형식의 일반 텍스트(.txt) 파일을 불러올 수 있습니다. `##`/`###` 제목은 선택입니다.

```
## Part 1. 제목 (선택)

### 섹션 제목 (선택)

* **English sentence.**
* 뜻: 한국어 뜻
* 발음: 한글 발음
```

## 기능
- **가리기 토글**: 영어/뜻/발음을 가리고 떠올리는 연습 (가린 글자 탭하면 잠깐 보임)
- **음성 듣기(TTS)**: 영어 문장을 원어민 발음으로 재생, 목소리 선택 가능
- **Part 필터 + 검색**
- **암기 체크 + 진행률 막대**: 전체/Part별 진도, 브라우저에 자동 저장
- **플래시카드 모드**: 한국어 뜻 → 탭하면 영어/발음 공개
- **다크/라이트 테마**
- **PWA**: 홈 화면에 추가하면 앱처럼 실행, 오프라인에서도 동작

## 파일 구성
| 파일 | 설명 |
|------|------|
| `index.html` | 연습 페이지 (콘텐츠 미포함) |
| `manifest.webmanifest`, `sw.js`, `icon.svg` | PWA 구성 파일 |
| `build_practice.py` | 앱 셸 생성 스크립트 |

```bash
python3 build_practice.py   # index.html 등 재생성
```
